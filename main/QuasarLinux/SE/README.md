# QuasarLinux SE (Second Edition)

Необычная редакция QuasarLinux, она не модульная, а один сплошной монолит.

Важные отличия от REV редакции: 


| Параметры | QuasarLinux-REV | QuasarLinux-SE  |
|-----------|-----------------|-----------------|
|  Ядро     | LTS/ZEN/VANILA  |    Blazar/ZEN   |
| Окружение |Зависит от профиля| Только Plasma  |
| Обновления| только пакеты   |    Image        |
| база      | Artix           |    QuasarLinux  |
